# Design Document

## Overview

The Styleguide / UI Kit page is an internal reference page for planner UI decisions. It should feel like part of the app rather than a marketing or documentation page.

## Layout

- Use the standard app shell and dotted-paper page background.
- Use the same content width rhythm as the working pages, with specimens grouped into quiet full-width sections.
- Avoid card-heavy presentation. Individual specimens can be framed only when the boundary helps comparison.

## Monthly Calendar Specimen

The current hardcoded Monthly calendar should move into this page as a component specimen. It should demonstrate:

- Weekday header row.
- May 2026 month grid.
- A few hardcoded task labels.
- Current-day emphasis.
- Planner-style muted borders and compact text.

## Navigation Icon Specimens

Show the main sidebar icon set:

- Daily: bullet journal dot.
- Inbox: chevron.
- Monthly: three bullet dots with three horizontal lines.
- Habits: repeat mark.

The Monthly icon should avoid a generic calendar glyph and instead read as a monthly index/list symbol.
